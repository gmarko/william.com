The Shift Register

The 74HC595 shift register was used to solve the problem of controlling 48 LED cathodes with a single Arduino chip. The shift register is used to store 8 bits of information and output it using 8 pins on the chip. It can be controlled by only three pins from a micro-controller and can be daisy-chained.

To push information into the register you use the Data and Clock pins to push a string of bits into the chip which can then be pushed on to the next chip connected in the chain. In the cube, I used a total of six registers to control all 48 cathodes coming from the cube.
