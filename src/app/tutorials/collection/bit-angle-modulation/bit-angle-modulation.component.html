arrow_back Tutorial: Bit Angle Modulation

Bit Angle Modulation (BAM) was the technique used to control all 192 LEDs in the 4x4x4 RGB LED Cube Project.

BAM works by setting a n-bit brightness value for each LED. For example, in a 4-bit example, the brightness can be set to 13, or 1101 in binary. Each bit has a value associated with it that is necessary for BAM.

Each iteration of BAM consists of n interruptions. (n as in n bit BAM) In the case of 15 degree brightness, each iteration consists of 15 interruptions. In each interruption we will run through each LED and turn them on according to their brightness value. In the case of the led cube, each interruption is executed by the arduino interrupt service routine.

To display the right-most bit, (0001) we will run through the first iteration and any LED with the last bit as a one will be turned on and off, and if it is a zero it will simply pause on the LED and skip it.

Then the next bit over (0010) will have two iterations, because it is a two-value bit. Then the next bit (0100) is run with four iterations, and eight iterations for the last bit. (1000)
